# 2025 年秋季计算机网络实验课编程大作业要求

## 一、提交要求

1. **截止时间**：第十五周周日（12 月 21 日）24:00 前，超期提交视为未提交。
2. **提交格式**
   - 组长以邮件发送包含所有小组成员作业的压缩文件夹，文件夹及邮件主题命名为：小组 x*编程大作业*已收齐/未交同学名字。
   - 个人作业命名：名字*学号*编程大作业，作业文件夹内需包含源代码文件、实验报告（作业一额外提交 exe 可执行程序），实验报告为 PDF 格式，内部格式无要求。

## 二、作业内容（二选一）

### （一）ping 程序实现

使用 Winsock 原始套接字编写 ping 控制台程序，需实现以下功能：

1. 支持指定目标主机 IP 或域名（如`ping www.baidu.com`）。
2. 每次发送 4 个 32B 的 ICMP 回送请求报文，发送和接收超时时间设为 1000ms。
3. 收到应答报文时，显示数据大小、源 IP、序号、响应时间；超时则显示“Request time out.”。
4. 最后展示用户名及统计信息（发出/收到报文数、丢包数、丢包率）。
5. 自行设计实现额外功能可加分。

### （二）爬取课表

1. 通过川大教务处学生登录系统，爬取所在班级课表并标准化存储结果。
2. 需考虑反爬虫机制及基于应用层知识的解决方法。
3. 自行设计实现额外实验内容可加分。
4. 课表存储结果需包含课程号、课序号、课程名、任课教师、学分、周次、上课时间、校区、教学楼、教室等信息（参考示例格式）。

## 三、注意事项

1. 无故缓交作业，成绩记为 50 分；抄袭则成绩记为 0 分。
2. 实验报告需详细记录实验步骤，搭配文字描述与截图（避免图片堆砌）。
3. 需记录并总结实验中的问题与解决办法。
4. 源程序需添加代码注释（如函数功能、关键代码说明），额外实现功能需补充对应描述。
5. 文档格式需规范，可使用模板或自拟格式（包含封面、目录、页眉页脚等），注意排版、字体、行距；可补充选题相关理论/拓展知识以加分。
6. 实验报告参考格式包含问题描述、基本要求、工具/准备、分析与实现、测试、思考与改进、总结、附件等板块，同时需绘制相关功能流程图（如 ping 程序的整体流程、功能控制模块流程）。
